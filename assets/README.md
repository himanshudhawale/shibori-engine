# Shibori Engine Brand Assets

## Icon

`shibori-engine-icon.svg` combines three ideas:

- the red circle and inner ring resemble a Japanese seal;
- the stacked bars represent structured database blocks;
- the opposing chevrons and narrower center block represent adaptive
  compression.

The icon is intentionally geometric so it remains recognizable at repository,
documentation, and command-line avatar sizes.

## Colors

| Name | Value | Use |
| --- | --- | --- |
| Shibori red | `#B72C32` | Seal and data marks |
| Rice paper | `#FFF8E8` | Ring, layers, and arrows |

The SVG includes an accessible title and description. Keep the original
`viewBox` and color contrast when producing derived sizes.
